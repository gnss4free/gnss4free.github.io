Приложение для автомобильной системы и мобильного телефона, построенных на операционной системе Android.

## Текущая функциональность

Первый релиз включает две функции:
1. **Автоматическая раздача интернета**
   - Активируется при подключении телефона к автомобилю через Bluetooth
   - Включает точку доступа (HotSpot) на телефоне

2. **Передача геолокации для навигации**
   - Активируется при регистрации в локальной сети системы-потребителя геолокации (как правило, автомобиля)
   - Система-провайдер геолокации (как правило, телефон с GNSS-чипом) рассылает данные о геолокации всем зарегистрированным системам-потребителям в локальной сети
   - Система-потребитель получая данные о геолокации делает их доступными Android-приложениям через функциональность ОС Android "Приложение для фиктивных местоположений"

## Ключевые особенности приложения 
#### 1. **Простой и лаконичный интерфейс**
   - Минималистичный дизайн
   - Интуитивно понятная настройка,
   - Не требуется вводить IP-адреса, названия сетевых интерфейсов и т.п.
![Интерфейс](/asset/images/RU_Light_ProviderInterface.jpeg){:width="30%" .align-center}

#### 2. **Надежная логика работы**
   - Использование **FSM** (Finite State Machine — конечный автомат) для управления состоянием приложения
   - Возможны разные варианты (конфигурации) использования
      - с ожиданием подключения по bluetooth и без
      - с включением точки доступа и без
   - Эффективный и предсказуемый алгоритм установления связи "Провайдер ↔ Потребитель":
       - Когда в локальной сети нет потребителя, передача данных прекращается, тем самым телефон не тратит свой разряд
       - Не используются "магические" значения для каких-либо параметров
       - Не используются временные задержки для достижения предполгаемого состояния системы

#### 3. **Максимальная производительность**
   - Эффективный двухсторонний **Бинарный протокол** (Минимальная задержка при передаче данных)
   - Актаульность данных не менее важна, чем точнось, поэтому на стороне системы-потребитя показываются временные параметры переданных данных, в том числе "честный" возраст данных
     
![Экран статистики](/asset/images/RU_Light_LowLatency.jpeg){:width="100%"}

#### 4. **Не требует вложений**
   - Программа бесплатна для частного использования
   - Не требуется приобретать дополнительные устройства/модули, выполнять аппаратную доработку автомобиля
   - Идеально, чтобы "попробовать"

## Планируемое расширение функциональности
  1. Передача буфера обмена от провайдера к потребителю
  2. Передача и установка приложений на системе потребителе
  3. Передача оповещений от выбранных приложений в системе провайдера в систему потребитель
  4. Автоматиеское обновление (OTA)
  5. Поддержка управления точкой тоступа на телефоне с ОС Andorid 14 и выше
  6. Поддержка других источников геолокации (USB-, Bluetooth-донглы, DIY GNSS Устройства)

#### Требования к системе
 - Телефон (система-потребитель) - Android 6.0 и выше, отключенная экономия энергии для приложения
 - Автомобиль (система-потребитель) - Android 6.0 и выше, доступ к разделу для разработчиков для установки "Приложения для фиктивных местоположений"

#### Ограничения
  - Управление точкой доступа осуществляется с использованием приложения-хелпера Macrodroid Connectivity Helper, поддерживаются ОС Android **до версии 14**.
  - На Android 6.0, может не работать скачивание и установка Macrodroid Connectivity Helper из-за устаревших корневых сертификатов в этой системе (сама система уже порядком устарела и не рекомендуется).
